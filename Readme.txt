# Student Management System (Java)

This is a simple **Java console-based project** to manage student records.

## Features
- Add Student
- View Students
- Update Student
- Delete Student

## How to Run
1. Open project folder in any IDE (Eclipse/IntelliJ/VS Code).
2. Compile both files:
   - `javac Student.java`
   - `javac StudentManagement.java`
3. Run:
   - `java StudentManagement`
